# 高度可自定义博客模版设计文档

> 日期：2026-06-06
> 状态：已批准

## 项目概述

一个高度可自定义的个人博客模版，部署在 Cloudflare Pages 上，支持管理员后台进行全方位定制。博客前台采用静态生成（SSG）+ 增量静态再生（ISR），后台采用动态交互，所有数据存储在 Cloudflare D1 数据库中。

## 需求总结

| 分类 | 需求 |
|------|------|
| 可自定义范围 | 主题/样式、功能模块、内容结构、SEO |
| 部署平台 | Cloudflare Pages |
| 数据库 | Cloudflare D1（5GB 免费 SQLite） |
| 图片存储 | Cloudflare Images + 外部图床（后台自由切换） |
| 费用 | 全部免费 |
| 管理员认证 | JWT Token + 账号密码登录 |
| 访问控制 | 只有管理员能进入后台 |

## 一、系统架构

### 技术栈

| 技术 | 用途 |
|------|------|
| Next.js 16 | 前端框架（App Router） |
| Cloudflare D1 | SQLite 数据库（存储内容、配置） |
| Cloudflare Workers | 后端 API 层（认证、数据操作） |
| Cloudflare Images | 图片存储（可选） |
| 外部图床 | 图片存储（可选，用户可切换） |
| Cloudflare Pages | 前台静态页面托管 |
| bcrypt + JWT | 认证与安全 |

### 架构图

```
用户浏览器
    │
    ├── 访问博客前台 ────→ Cloudflare CDN ────→ 静态页面（SSG/ISR）
    │                                              │
    │                                              └── D1 数据库（内容/配置）
    │
    └── 访问管理后台 ────→ Next.js SSR ────→ Workers API
                                │                    │
                                └── 客户端交互 ────→ D1 数据库
                                                    │
                                                    └── Cloudflare Images / 图床
```

### 数据流向

- **博客前台**：用户请求 → CDN 缓存 → 静态页面（内容来自 D1 的预生成页面）
- **管理后台**：管理员操作 → Workers API → D1 数据库（实时读写）
- **内容更新**：管理员提交 → Workers API 写入 D1 → 触发 ISR 重新生成受影响的静态页面

### Cloudflare 免费层级评估

| 服务 | 免费额度 | 个人博客够用吗？ |
|------|---------|----------------|
| Cloudflare Pages | 不限站点/带宽，500分钟构建/月 | ✅ 完全够用 |
| Cloudflare Workers | 100,000 次请求/天 | ✅ 个人博客利用率 < 1% |
| Cloudflare D1 | 5GB 存储 | ✅ 可存约 100 万篇纯文字文章 |
| Cloudflare Images | 1GB 存储，10万次请求/月 | ✅ 可存约 1600 篇图文文章 |
| Cloudflare SSL | 免费 | ✅ 自动配置 |

## 三、前台页面结构

| 页面 | 路由 | 渲染方式 |
|------|------|---------|
| 首页 | `/` | SSG + ISR |
| 文章详情 | `/posts/[slug]` | SSG + ISR |
| 分类页 | `/categories/[slug]` | SSG + ISR |
| 标签页 | `/tags/[slug]` | SSG + ISR |

| 友链页 | `/friends` | SSG + ISR |

### 各页面详细功能

**首页**：
- 文章列表（瀑布流/卡片布局）
- 支持分页（每页 10-20 篇）
- 显示精选文章
- 侧边栏模块（可配置显示/隐藏、排序）

**文章详情页**：
- 文章标题、发布时间、分类、标签
- 封面图片（可配置是否显示）
- Markdown 富文本渲染（代码高亮、图片灯箱）
- 上一篇/下一篇导航
- 文章目录/导航（Table of Contents）

**分类页/标签页**：
- 该分类/标签下的所有文章列表
- 支持分页

**归档页**：
- 所有文章按年份/月份组织
- 时间线样式展示

**关于页/友链页**：
- 自定义页面模板

**404 页面**：
- 自定义 404 页面样式

## 六、部署架构

### 项目结构

```
blog-template/
├── src/
│   ├── app/
│   │   ├── (blog)/              # 博客前台
│   │   ├── admin/               # 管理后台
│   │   └── api/                 # API 路由
│   ├── components/
│   ├── lib/
│   └── types/
├── prisma/
│   └── schema.prisma
├── functions/                   # Workers Functions
├── wrangler.toml                # Workers 配置
├── wrangler.jsonc               # Pages 配置
└── next.config.ts
```

### 部署步骤

1. 安装依赖并构建：`npm install && npm run build`
2. 创建 D1 数据库：`wrangler d1 create blog-template-db`
3. 执行迁移：`wrangler d1 migrations apply blog-template-db`
4. 部署到 Cloudflare Pages（推荐 GitHub 集成自动部署）
5. 配置环境变量（JWT_SECRET 等）
6. 绑定自定义域名（SSL 自动配置）

## 七、安全与权限控制

### 认证安全

- 密码使用 bcrypt（cost factor 12）加密
- JWT 有效期 7 天，过期需重新登录
- 登录失败限制：同一 IP 5 分钟内连续 5 次失败，锁定 15 分钟
- 错误信息不返回密码提示

### 权限控制

- **公开接口**：无需认证（博客前台数据读取）
- **管理员接口**：所有 POST/PUT/DELETE 操作需要管理员 Token
- **单一管理员**：系统只有一个管理员账号

### API 安全

- 输入校验：所有参数进行 Schema 校验
- SQL 注入防护：参数化查询
- XSS 防护：HTML 转义
- 请求频率限制：同一 IP 同一接口 1 分钟最多 60 次
- CORS：只允许来自博客域名的跨域请求

### 数据安全

- 密码加密存储，数据库不存明文
- JWT Secret 存储在 Cloudflare 环境变量中
- 文件上传限制：仅图片、最大 5MB
- 外部链接渲染时添加 `rel="noopener noreferrer"`
